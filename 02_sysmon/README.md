# Sysmon Endpoint Telemetry

## Purpose

This folder documents Microsoft Sysmon setup, Windows endpoint telemetry collection and event analysis notes for SOC analyst practice.

## Skills Covered

- Sysmon installation verification
- Sysmon Operational log review
- Endpoint telemetry collection
- Event ID analysis
- Process activity review
- Parent-child relationship analysis
- Command-line field review

## Important Sysmon Concepts

| Area | Analyst Focus |
|---|---|
| Process activity | Understand what executed on the endpoint |
| Parent process | Identify what launched the activity |
| Command line | Review execution context |
| User | Identify account context |
| Timestamp | Build investigation timeline |

## SOC Analyst Value

Sysmon provides rich endpoint visibility. SOC analysts use Sysmon telemetry to investigate process behavior, suspicious execution, persistence indicators and endpoint activity timelines.
