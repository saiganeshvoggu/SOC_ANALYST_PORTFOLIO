# Event ID 4634 – Logoff

## Description

Event ID 4634 indicates that a user session has been terminated.

## Importance

This event helps analysts understand user session timelines and authentication activity.

## Investigation Use Cases

* Session tracking
* User activity monitoring
* Authentication timeline creation

## Lab Findings

Event ID 4634 was observed in the Windows Security Log following user logoff activity.
