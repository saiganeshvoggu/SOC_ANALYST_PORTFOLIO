# Workbooks

## Overview

Workbooks in Microsoft Sentinel provide interactive dashboards and visualizations that help SOC analysts monitor security data, identify trends, and investigate threats.

Workbooks transform raw log data into charts, graphs, maps, and tables.

---

## Purpose

Workbooks help:

* Visualize security data
* Monitor threats
* Track incidents
* Analyze trends
* Create executive reports
* Improve situational awareness

---

## Workbook Workflow

Data Source

↓

Log Analytics Workspace

↓

KQL Query

↓

Workbook

↓

Visualization

---

## Types of Visualizations

### Tables

Display structured security data.

Example:

Top Failed Login Users

---

### Charts

Display trends over time.

Example:

Failed Logins Per Day

---

### Pie Charts

Show percentage distribution.

Example:

Incident Severity Distribution

---

### Maps

Display geographic locations.

Example:

Login Attempts by Country

---

### Graphs

Show relationships and activity trends.

## Interview Questions

### What are Workbooks?

Workbooks are interactive dashboards used to visualize security data in Microsoft Sentinel.

### Why are Workbooks important?

They help analysts quickly understand trends, incidents, and security events.

### What data can Workbooks display?

Logs, alerts, incidents, threat intelligence, user activity, and endpoint telemetry.
