# Microsoft Sentinel

## Overview
Microsoft Sentinel is a cloud SIEM platform used by SOC teams for log collection, analytics rules, incident investigation, threat hunting, automation, and dashboards.

This folder will contain Microsoft Sentinel learning notes, KQL examples, incident investigation workflows, and future lab documentation.

- KQL basics
- MITRE ATT&CK mapping in Sentinel

## SOC Analyst Relevance
Microsoft Sentinel is widely used in enterprise security operations. Learning Sentinel helps demonstrate SIEM investigation skills, KQL querying, cloud monitoring, and incident response workflow knowledge.
