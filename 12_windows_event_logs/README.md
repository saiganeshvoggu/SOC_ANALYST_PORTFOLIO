# Windows Event Logs

## Purpose

This folder documents Windows Security Event Log investigation notes and common event IDs used in SOC analysis.

## Skills Covered

- Windows Event Viewer navigation
- Security Event ID review
- Authentication event analysis
- Privileged logon review
- Event timeline building
- Log-based investigation notes

## Important Event IDs

| Event ID | Meaning |
|---|---|
| 4624 | Successful logon |
| 4625 | Failed logon |
| 4634 | Logoff |
| 4672 | Special privileges assigned |
| 4688 | Process creation, if enabled |
| 1102 | Security log cleared |

## SOC Analyst Value

Windows Event Logs are core evidence for authentication review, endpoint investigation and incident response.
