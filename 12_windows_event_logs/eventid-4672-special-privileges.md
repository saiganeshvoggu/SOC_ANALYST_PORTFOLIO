# Event ID 4672 – Special Privileges Assigned

## Description

Event ID 4672 is generated when special privileges are assigned to a new logon session.

## Importance

This event is commonly associated with:

* Administrator accounts
* Privileged service accounts
* Elevated sessions

## Investigation Focus

SOC Analysts review Event ID 4672 to identify:

* Administrative logins
* Privilege escalation activity
* High-risk account usage

## Lab Findings

Event ID 4672 was observed immediately after successful authentication events.

## Detection Use Cases

* Privileged account monitoring
* Privilege escalation investigations
* Administrative activity tracking
