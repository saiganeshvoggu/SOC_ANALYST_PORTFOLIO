# Threat Intelligence and IOC Analysis

## Purpose

This folder documents threat intelligence basics, IOC extraction, reputation checks and external enrichment workflows used during SOC investigations.

## Skills Covered

- Indicator extraction
- Domain and URL review
- IP and hash reputation checks
- VirusTotal usage
- URLScan.io usage
- Analyst verdict documentation

## IOC Types

| IOC Type | Examples |
|---|---|
| Domain | Suspicious sender or linked domain |
| URL | Link found in email or alert |
| IP address | Source or destination IP |
| File hash | Malware or attachment hash |
| Email address | Sender or reply-to address |

## SOC Analyst Value

Threat intelligence helps analysts enrich alerts, validate suspicious indicators and support incident response decisions.
