I have hands-on experience with Microsoft Sentinel, Splunk, Sysmon, Windows Event Logs, KQL, threat hunting and incident documentation. ## Why SOC Analyst?

I like investigation-based work where I can analyze alerts, understand security events, review logs and help protect systems. SOC analyst work combines technical analysis, documentation and real-time decision-making, which matches my interest in cybersecurity operations.

---

## Explain your SOC portfolio project

My SOC portfolio is a hands-on lab built around Splunk, Sysmon, Microsoft Sentinel, Azure Arc, Azure Monitor Agent, Data Collection Rules, Windows Security Events and KQL. I generated and investigated Windows security events such as failed logons, successful logons, privileged logons, process creation, PowerShell execution, service installation, scheduled task creation, user creation, group membership changes and Security log clearing. I documented detections, threat hunts, incident investigations and SOC case management workflows.

I bring hands-on SOC lab experience, previous SOC operations exposure and strong documentation discipline.
