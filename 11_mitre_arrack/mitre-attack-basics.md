# MITRE ATT&CK Basics

## Purpose
MITRE ATT&CK is a knowledge base of adversary tactics and techniques used by cybersecurity professionals to understand, detect, and respond to attacks.

## Key Tactics
- Initial Access
- Execution
- Persistence
- Privilege Escalation
- Defense Evasion
- Credential Access
- Discovery
- Lateral Movement
- Collection
- Command and Control
- Exfiltration
- Impact
