# Incident Response

## Purpose

This folder documents incident response playbooks, escalation templates and investigation workflows for SOC operations.

This section represents work aligned with my NationsBenefits SOC Analyst experience.

---

## Skills Covered

- Initial triage
- Scope validation
- Evidence collection
- Severity classification
- Containment planning
- Escalation communication
- Closure documentation
- Lessons learned notes

---

## IR Workflow

```text
Detect
  -> Triage
  -> Investigate
  -> Scope
  -> Contain
  -> Escalate
  -> Document
  -> Close
```

- Endpoint security alert validation
