# SOC SOPs and Runbooks

## Purpose

This folder contains standard operating procedures, analyst checklists and response workflows for common SOC activities.

## Topics Covered

- Alert triage
- Failed login review
- Email quarantine review
- Endpoint investigation notes
- BitLocker recovery workflow
- Escalation process
- Evidence collection standards

## SOP Format

```text
Purpose
Trigger
Steps
Evidence Required
Escalation Criteria
Closure Notes
```

## SOC Analyst Value

SOPs help analysts respond consistently, document actions clearly and escalate issues with the right context.
