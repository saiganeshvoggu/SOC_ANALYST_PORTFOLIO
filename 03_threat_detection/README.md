# Threat Detection

## Purpose

This folder documents threat detection concepts, Windows Event ID monitoring, SIEM detection ideas and MITRE ATT&CK mapping notes.

---

## Skills Covered

- Suspicious event review
- Event ID mapping
- Detection logic planning
- Alert rule documentation
- MITRE ATT&CK alignment
- Analyst investigation notes
- False positive considerations

---

## Detection Areas

| Area | Example |
|---|---|
| Authentication | Failed login, successful login, privileged logon |
| Endpoint | Process activity, PowerShell review, parent-child process review |
| Account changes | New user, group membership change |
| Service and task activity | Service installation, scheduled task creation |
| Log review | Security log cleared |

---

## SOC Analyst Value

Threat detection work shows the ability to convert logs into useful monitoring and investigation ideas. This folder supports the detection engineering and threat hunting phases.
