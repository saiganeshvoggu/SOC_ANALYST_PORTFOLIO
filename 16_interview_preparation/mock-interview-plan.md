# Mock Interview Plan

## Objective

Prepare for SOC Analyst interviews using the completed portfolio labs.

---

## Round 1 - Introduction and Resume

Focus:

- Tell me about yourself
- Explain SOC experience
- Explain portfolio project
- Why SOC Analyst

---

## Round 2 - Technical Fundamentals

Focus:

- Splunk
- Sysmon
- Windows Event Logs
- MITRE ATT&CK
- Microsoft Sentinel
- KQL

---

## Round 3 - Scenario-Based SOC Questions

Focus:

- Failed login alert
- PowerShell alert
- Service creation alert
- Scheduled task alert
- Security log cleared alert
- New user account alert
- Group membership change alert

---

## Round 4 - Practical KQL Questions

Focus:

- Find failed logins
- Count events by account
- Sort latest events
- Hunt PowerShell execution
- Review service and scheduled task activity

---

## Round 5 - HR and Behavioral

Focus:

- Communication
- Shift readiness
- Handling pressure
- Documentation habits
- Learning attitude

---

## Final Goal

Be ready to explain practical SOC work using examples from the GitHub portfolio.
