# SOC Case Management Phase

## Status

**Phase Status:** Completed  
**Focus:** SOC ticket lifecycle, severity classification, escalation and closure documentation  
**Platform Context:** Microsoft Sentinel / Azure Log Analytics  
**Environment:** Personal SOC lab

---

## Objective

This phase documents practical SOC case management workflows. The goal was to convert alert investigation results into professional SOC ticket notes, severity decisions, escalation decisions and closure documentation.

---

## Case Management Workflow

```text
Alert Received
  -> Ticket Created
  -> Initial Triage
  -> Evidence Review
  -> Severity Assignment
  -> Escalation Decision
  -> Containment / Recommendation
  -> Documentation
  -> Closure
```
