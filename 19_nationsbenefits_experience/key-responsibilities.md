# Key Responsibilities

## Role

Associate SOC Analyst

## Core Responsibilities

- Reviewed security alerts from service desk and security platforms.
- Supported investigation of account-related security events.
- Assisted with endpoint review and device security checks.
- Maintained tracking sheets for security activities and operational follow-ups.
- Supported DLP review activities using Microsoft Purview concepts and workflows.
- Assisted with email security and quarantine review activities.
- Participated in vulnerability review discussions with senior security team members.
- Helped prepare security documentation, process notes, and response guides.
- Escalated selected items to Tier-2 analysts and Information Security Engineers.

## Environment Scale

Supported a 2,000+ user enterprise environment across India and the United States.
