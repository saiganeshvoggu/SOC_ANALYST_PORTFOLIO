# MITRE ATT&CK Mapping

## Purpose

This folder documents MITRE ATT&CK tactics, techniques and mapping notes for SOC investigations.

## Skills Covered

- Understanding tactics and techniques
- Mapping alerts to attacker behavior
- Connecting logs to MITRE techniques
- Writing analyst-friendly detection notes
- Supporting interview explanations

## Covered Techniques

| Technique | Name | SOC Relevance |
|---|---|---|
| T1566 | Phishing | Email security investigations |
| T1059 | Command and Scripting Interpreter | Command execution review |
| T1003 | Credential Dumping | Credential access investigation |
| T1078 | Valid Accounts | Suspicious login analysis |

## SOC Analyst Value

MITRE ATT&CK mapping helps analysts explain attacker behavior, organize detections and communicate investigation context clearly.
